# [22. Generate Parentheses](https://leetcode.com/problems/generate-parentheses/)

Medium

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

Example 1:

```s
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```

Example 2:

```s
Input: n = 1
Output: ["()"]
```

Constraints:

- 1 <= n <= 8
